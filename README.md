# cappuccino

Simple image viewer

# Requirements

* beautifulsoup4
* Pillow
* PyQt5
* sip

# How to

Download image and Display
```
pythin cappuccino.py image_keyword
```

Display of the downloaded image
```
pythin cappuccino.py
```

# Pyinstaller

```
pip install pyinstaller
pyinstaller cappuccino.py --onefile --noconsole --clean
```
